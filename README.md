# eFarming

eFarming is a full-stack Django web application designed to bridge the gap between local farmers and consumers. It eliminates middlemen by providing a direct-to-consumer marketplace for fresh produce.

## Features
* Direct marketplace for local produce
* Order tracking system
* Integrated agricultural services and bulk discounts

## Technologies Used
* **Backend:** Python, Django
* **Database:** SQLite (default for development)
* **IDE:** PyCharm
